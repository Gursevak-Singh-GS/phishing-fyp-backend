Phishing Detection System with Explainable AI (XAI)Project OverviewThis project implements a hybrid machine learning framework to detect phishing URLs with high accuracy and transparency. The system evaluates URLs using two primary models: a Logistic Regression baseline and an optimized Random Forest classifier. By integrating Explainable AI (XAI), the system doesn't just block a site—it explains why based on live structural and lexical analysis.  Key FeaturesDual-Model Analysis: Includes both the baseline Logistic Regression and the high-performance Random Forest model.  High Accuracy: The Random Forest model achieves a 0.9719 F1-Score.  Live Feature Extraction: app_final.py processes live DNS and HTML metadata to detect zero-day threats.  Visual Evidence: Comprehensive plots including ROC curves, confusion matrices, and feature importance.  Installation and Setup1. Backend Setup (phishing-backend)Navigate to directory: cd phishing-backendInstall dependencies: pip install -r requirements.txt  Run the Final Application: Execute python app_final.py  Note: Ensure lr_model.pkl and rf_model.pkl are in this folder for the app to function.2. Frontend Setup (phishing-frontend)Navigate to directory: cd phishing-frontendInstall dependencies: npm install  Run UI: npm startProject File Structure (Backend)app_final.py: The main entry point for the detection API.  lr_model.pkl & rf_model.pkl: Serialized baseline and optimized models.  dataset.csv: The training dataset used for model evaluation.  compare_models.py: Script used to generate performance comparisons.  Visual Performance AssetsYour backend folder includes the following evidence used in the Supporting Document:model_comparison.png: High-level comparison of LR vs. RF.  rf_roc_curve.png: Evidence of the 0.99 AUC performance.  rf_feature_importance.png: Shows which URL features drive the XAI engine.  rf-terminal.png: Direct evidence of the training accuracy (96.79%).  Usage InstructionsLaunch the backend using app_final.py.Access the React UI on http://localhost:3000.Input a URL to see the comparison between the baseline and optimized results, along with the XAI-driven security alert.
